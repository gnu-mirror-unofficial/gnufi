Introduction to GNUFI

GNUFI is designed to be a firmware compatible with the Unified Extensible Firmware Interface.

Downloading GNFI

At this point there has no been any official release of GNUFI. The version controlled source repository can be accessed using Bazaar: 
  bzr branch http://repo.ginandtonique.org/gnufi/gnufi.dev

Mailing Lists/Newsgroups

The main discussion list is <bug-gnufi@gnu.org>, and is used to discuss all aspects of GNUFI, including its development and porting.

To subscribe to the GNUFI mailing list, send an empty mail with a Subject: header line of just "subscribe" to the bug-gnufi-request list.

Another way to subscribe is visiting the web page bug-gnufi.

Wiki

There is a wiki available for the developers and users for GNUFI.

Maintainer
