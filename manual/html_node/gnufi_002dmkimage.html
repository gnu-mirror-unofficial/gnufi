2.3 The `gnufi-mkimage' Utility

The `gnufi-mkimage' utility takes a kernel script file as argument (See Kernel scripts.) and creates a firmware image based on its contents. The utility also takes the following options;

-o FILE
Specify output file.
-d DIR
Specify where modules specified in the configuration file can be found.
